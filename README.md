# AssistantSG

Poker assistant (Texas Hold'em 9-max) con:
- Setup parametrico e persistente (SharedPreferences)
- Preflop ranges (Raise/Call/Fold) per posizione
- Equity MonteCarlo preflop/flop/turn/river
- Postflop: se POT/BET vuoti → modalità veloce solo equity (non vincolante)

Build APK via Codemagic.
